# MILLIE Pi install

See the full guide: [millie-pi/README.md](../millie-pi/README.md)

## TL;DR

On your Raspberry Pi:

```bash
cd MILLIE/millie-pi
bash setup.sh
~/.millie-pi-install/run-millie-pi.sh
```

On your phone: MILLIE running + scanning. Same WiFi as the Pi (or USB tether).

Pi dashboard: **http://\<pi-ip\>:8780/**

Phone must run a CIVOPS build with `/api/hub/handshake` (included in current tree).
